## What is EQX?

EQX is a custody-and-settlement platform that lets businesses accept stablecoin payments and settle them as local fiat currency — across Kenya, Nigeria, Ghana, South Africa, and more.

You send USDC. Your customers receive KES, NGN, GHS, or ZAR directly to mobile money or bank accounts. EQX handles custody, liquidity, compliance, and payout routing in between.

## Who is this API for?

The EQX Settlement API is designed for **integrators**: fintech platforms, payment processors, neobanks, and remittance services that want to embed African payout infrastructure into their products.

If you're an individual end-user sending a single payment, you don't need this API — use the EQX app directly.
