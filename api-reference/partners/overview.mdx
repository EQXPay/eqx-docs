---
title: Partners
description: Workspace management, memberships, and invitations
---

The Partners API manages your integrator workspace — team members and pending invitations.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/partner/account` | Get the authenticated partner account |
| `GET` | `/api/partner/memberships` | List workspace members |
| `POST` | `/api/partner/memberships` | Add a member (existing EQX user) |
| `PATCH` | `/api/partner/memberships/:id` | Update a member's role |
| `DELETE` | `/api/partner/memberships/:id` | Remove a member |
| `GET` | `/api/partner/invitations` | List pending invitations |
| `POST` | `/api/partner/invitations` | Send an invitation |
| `POST` | `/api/partner/invitations/:id/accept` | Accept an invitation |
| `POST` | `/api/partner/invitations/:id/cancel` | Cancel an invitation |

## Roles

| Role | Permissions |
|------|-------------|
| `OWNER` | Full access including billing and security settings |
| `ADMIN` | Full operational access, cannot change billing |
| `DEVELOPER` | API access, cannot manage members or billing |
| `VIEWER` | Read-only access |
