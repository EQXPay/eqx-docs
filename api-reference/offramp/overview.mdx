The Offramp API provides the full quote-and-commit flow for converting stablecoins to local fiat.

See [Off-Ramp concepts](/concepts/off-ramp) for the flow overview.

For payout success expectations, provider dependency behavior, and final-state guidance, see [Reliability expectations](/concepts/off-ramp#reliability-expectations).

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/routes` | List available offramp corridors |
| `GET` | `/api/transactions` | List payout transactions (paginated) |
| `POST` | `/api/quotes` | Create a locked quote |
| `POST` | `/api/transactions/commit` | Commit a quote into a live payout |
| `GET` | `/api/transactions/:id` | Get transaction detail |
